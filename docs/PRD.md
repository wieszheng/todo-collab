# 产品需求文档 (PRD)

## 项目概述

**Todo Collab** 是一个现代化的待办事项协作平台，支持个人任务管理和团队协作。

## MVP 核心功能

### 1. 用户管理
- 邮箱注册/登录
- 基本个人信息管理

### 2. 任务管理
- 任务创建（标题、描述、截止日期、优先级）
- 任务列表查看（列表视图）
- 任务编辑/删除
- 任务状态切换（待办/进行中/已完成）
- 任务优先级设置（高/中/低）

### 3. 协作功能
- 任务分配给其他用户
- 查看分配给我的任务

### 4. 通知系统
- 任务到期前1天邮件提醒
- 任务分配邮件通知

## 用户故事

### US-01 用户注册
- **As a** 新用户
- **I want to** 通过邮箱快速注册账号
- **So that** 我可以开始使用平台管理任务

### US-05 创建任务
- **As a** 用户
- **I want to** 创建任务并设置标题、截止日期、优先级
- **So that** 我可以记录和追踪待办事项

### US-12 分配任务
- **As a** 项目管理员
- **I want to** 将任务分配给特定团队成员
- **So that** 责任明确，任务能够被执行

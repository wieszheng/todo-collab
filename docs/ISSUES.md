## ✅ 已修复

### 1. 登录错误密码没有提示
- **现象**: 输入错误密码后，页面没有显示任何错误提示
- **预期**: 显示"邮箱或密码错误"等提示
- **模块**: 前端 - LoginPage.tsx
- **修复**: `api.ts` 响应拦截器在登录页跳过 401 拦截，让页面自己处理错误
- **状态**: ✅ 已修复 (2026-03-12)

### 2. 修改头像的裁剪框展示问题
- **现象**: 修改头像时裁剪框显示异常
- **模块**: 前端 - ProfilePage.tsx / AvatarUpload.tsx
- **修复**: 重构裁剪组件，正确定位裁剪框，支持拖拽调整
- **状态**: ✅ 已修复 (2026-03-12)

### 3. 头像上传弹窗显示异常
- **现象**: 上传图片后弹窗打开，但图片不显示，裁剪框不可见
- **模块**: 前端 - AvatarUpload.tsx
- **修复**: 重构布局，图片和裁剪框放在同一个容器中，确保定位基准一致
- **状态**: ✅ 已修复 (2026-03-12)

### 4. 调整头像弹窗被侧边栏遮挡
- **现象**: 裁剪弹窗被左侧侧边栏遮挡
- **根因**: 弹窗 z-index 比侧边栏低
- **修复**: 使用 React Portal 把弹窗渲染到 body 层级
- **状态**: ✅ 已修复 (2026-03-12)

### 5. 保存头像失败
- **现象**: 点击保存头像按钮后无反应
- **根因**: `userService.ts` 硬编码端口 8000，但后端运行在 8001
- **修复**: 改用统一的 axiosInstance，读取正确的 baseURL
- **状态**: ✅ 已修复 (2026-03-12)

### 6. 任务列表没有高度限制
- **现象**: 数据过多时列表超出页面
- **修复**: 添加 `max-h-[calc(100vh-320px)]` 和 `overflow-y-auto`
- **状态**: ✅ 已修复 (2026-03-12)
